# つかいかた

論理式を打ち込むように要求されるので、論理式を打ち込みます。

命題記号は必ず**Aから順番に大文字**で使用してください。
また、余計なカッコは使わないようにしてください。

使える記号は、
- × : 矛盾
- ∧ : かつ
- ∨ : または
- → : ならば
- ¬ : 否定
- A ~ Z : 命題記号
- () : 優先順位
です。

使用例をtautology_checklist.txtに載せておくので、そこでなんとなく分かってくれると嬉しいです。
